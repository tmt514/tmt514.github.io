## 我自己的習慣

我的寫法是閉區間的寫法，這種寫法的理論基礎是說：

```cpp
while(l <= r) {
    m = (l + r) / 2;
    if (a[m] <= target) {
        l = m + 1;
    } else {
        r = m - 1;
    }
}
```
